
- Analytical review of Birkenstock’s financial performance (2024–2025) with a peer comparison vs. ON Holding.

- Processing done in Power Query (M); visualizations built in Power BI with custom DAX and animated elements.

- All figures converted to EUR for consistency.

- Brand-consistent color schemes applied for both Birkenstock and ON.

- Navigation tip:

       -  Lock objects after opening the PBIX (View → Lock objects).

       - Toggle between companies via the interactive switch (Ctrl + Click).
